This sample illustrates how to change the visual appearance of our jQuery Rating control entirely using its 'cssVotes' property. It gives you the ability to apply different styles on any of the control's items. At the same time, for every one of the items you can set a different style for the following states: normal, selected and hovered. The second sample shows you how to select only one item, which is not supported by the jQuery Rating control, but can be achieved using our 'cssVotes' property.

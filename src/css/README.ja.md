==== structure ====

このフォルダーには、Ignite UI for jQuery コンポーネントの構造 （幅、高さ、マージン、パディングなど） に関連するすべてのスタイルが含まれています。このフォルダーのファイルは Ignite UI for jQuery コントロールが正しく動作する上で非常に重要です。

注: structure ファイルはどのテーマにも関連していません。
